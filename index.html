Domain Belum Dibenarkan (Authorized Domain)

Firebase menyekat log masuk daripada domain pratinjau ini. Sila daftarkan domain berikut ke dalam Firebase Console anda untuk membolehkan Log Masuk DELIMa/Google:

mengira_domain...
1. Log masuk ke Firebase Console.
2. Pilih projek sps-skb-d8256.
3. Buka menu Authentication -> tab Settings -> Authorized domains.
4. Klik Add domain, tampal domain di atas dan klik Simpan.
